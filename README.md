
## Installation

To install `mcp-server` using [uvx](https://github.com/astral-sh/uv), run:

uvx --from git+https://github.com/moritzvonangernIBM/mcp-test-deployment.git mcp-server


```json
{
  "mcpServers": {
    "Prompt": {
      "command": "uvx",
      "args": [
        "--from",
	"git+https://github.com/moritzvonangernIBM/mcp-test-deployment.git",
	"mcp-server"
      ]
    }
  }
}
````
